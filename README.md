## About

Simple server to app [Bexs Challenge](https://github.com/brunoJSX/bexs-challenge).

## Technologies

- [Express](https://expressjs.com/)

## Requirements

- Node ^12.22.1
- Yarn ^1.22.1

## Getting Started
 
Install the dependencies:

```shell
yarn
```
Starting the server:

```shell
yarn server
```

The server will start on `http://localhost:3333`
